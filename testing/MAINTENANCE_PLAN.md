# Maintenance Plan

## Weekly

- Review crash reports.
- Check Play Console vitals.
- Test login and sync.
- Test export to Gallery.
- Review user feedback.

## Monthly

- Update Android dependencies.
- Review AdMob policy messages.
- Review Play Billing status.
- Backup production database.
- Test restore from backup.
- Review legal policy pages.

## Before every release

- Run full QA test plan.
- Build signed AAB.
- Test via internal testing.
- Confirm privacy policy and Data Safety form still match app behavior.
- Confirm no test ad IDs in production release.
